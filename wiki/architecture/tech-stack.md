# Tech Stack

## Frontend
- **Vanilla HTML/CSS/JS**: No framework, single-file HTML pages
- **CSS Variables**: Custom properties for theming (--primary, --accent, etc.)
- **Socket.io Client**: Real-time communication with server

## Backend
- **Node.js**: Runtime environment
- **Express 5.x**: Web server framework
- **Socket.io**: WebSocket library for real-time multiplayer
- **SQLite3**: Embedded database for persistence

## Blockchain (Planned/Partial)
- **Solana Web3.js**: Solana blockchain interaction
- **@solana/spl-token**: SPL token operations
- **@metaplex-foundation/js**: NFT minting capabilities
- **Phantom Wallet**: Browser wallet integration

## External APIs
- **CoinGecko API**: Cryptocurrency market data and images

## Hosting
- **GitHub Pages**: Static frontend hosting
- **Node.js Server**: Backend for multiplayer (requires separate hosting)
