# Old Files

The `old/` directory contains legacy/backup versions of HTML files.

## Contents

| File | Description |
|------|-------------|
| `pogsthegame copy.html` through `copy 10.html` | Iterative versions of the game client |
| `index-orig.html` | Original landing page |
| `index-btest1.html`, `index-btest2.html` | Test/beta versions of landing page |
| `testb1.html` | Miscellaneous test file |
| `soltokenbalancechecker.html` | Solana token balance checking utility |

## Purpose

These files appear to be backup copies made during development. They track the evolution of the game client through multiple iterations.

## soltokenbalancechecker.html

A standalone utility for checking Solana token balances - separate from the main game functionality.
