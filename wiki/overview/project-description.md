# Project Description

CryptoPogs is a nostalgic web game that combines the classic 90s game of Pogs with cryptocurrency themes. Players collect, trade, and play with digital pogs representing various cryptocurrencies.

## What It Does

- **Collection**: Users can collect pogs featuring different cryptocurrencies (Bitcoin, Ethereum, Dogecoin, etc.)
- **Trading**: Trade pogs with other players
- **Gameplay**: Play the classic pogs game with a slam mechanic where face-up pogs are collected
- **Multiplayer**: Real-time PvP gameplay via Socket.io, plus single-player vs CPU mode
- **Wallet Integration**: MetaMask/Phantom wallet connectivity for authentication

## Target Platform

The application is a web-based game hosted on GitHub Pages with a Node.js/Express backend for multiplayer functionality.
